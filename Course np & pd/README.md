# NumPy & Pandas Fundamentals

## Overview

This notebook is a practice exercise using NumPy and pandas for data manipulation and visualization.

The purpose of this notebook is to solidify the fundamentals of data wrangling in Python using `NumPy` and `pandas` before moving on to building machine learning models. It serves as a training space to master concepts such as `reshape`/`melt`, aggregation operations, and data transformations, which are often the bottlenecks in any data project. The added value of this exercise is to minimize early errors and increase the speed and efficiency of preparing data correctly for future reuse.
