# Machine Learning Projects Collection

## Overview
This repository contains a curated collection of machine learning case studies implemented as Jupyter notebooks. Each folder presents a focused project with dataset exploration, preprocessing, model training, evaluation, and visualization.

## Projects Included
- Adult Income Prediction System
- Bank Campaign Success Predictor
- Bank Term Deposit Prediction System
- California Housing Price Predictor
- Diamond Price Predictor
- Healthcare Insurance Cost Predictor
- Handwritten Digit Recognition System
- Retail Sales & Customer Purchase Pattern Analysis
- Retail Sales Forecasting System
- Spotify Audio Feature-Based Recommendation System
- Stroke Risk Prediction System
- Taxi Fare Prediction System
- Titanic Survival Predictive Analytics
- Weather Forecasting System using Machine Learning
- Unsupervised Learning and Recommendations
- Exploratory Data Analysis with Matplotlib & Seaborn
- Course np & pd

## Common Techniques
- Data ingestion with pandas and CSV dataset loading
- Missing-value handling, categorical encoding, and scaling
- Supervised learning with Decision Trees, Random Forests, Logistic Regression, KNN, Linear Regression, and XGBoost
- Unsupervised learning with KMeans, DBSCAN, PCA, and t-SNE
- Evaluation with accuracy, precision/recall/F1, R², RMSE, silhouette score, and confusion matrices
- Visualization using Matplotlib and Seaborn for exploratory analysis and model diagnostics

## Usage
1. Install Python 3.8+ and required packages.
2. Install dependencies:
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn xgboost opendatasets
   ```
3. Open the desired notebook in Jupyter Notebook or JupyterLab.
4. Execute cells sequentially to reproduce the analysis and model training.

## Notes
- Many notebooks are educational and exploratory, designed to demonstrate common ML workflows.
- Several projects include multiple notebooks for model comparison.
- Check each project README for dataset requirements and key performance metrics.
